Technician Tool

A PC technician tool crafted with C# using the System and System Diagnostics library to make system changes and display system info

My Role: Using skills learned in C# and .NET to create a tool that runs natively on Windows and allows the user to run several useful system commands, run scripts, and programs.

Problem: Long helpdesk ticket times, new technicians not knowing essential system commands to make system changes or view information.

Solution: An Exe file that is capable of executing 10 essential system commands almost instantly. Reduced ticket times, ease-of-use for new technicians.
